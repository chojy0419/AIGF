Introduction
This project is an intelligent digital companion assistant built upon the architecture of the original Bella project. It is designed to transcend simple task automation, focusing on cultivating a warm, emotionally intelligent digital personality that grows and evolves alongside the user. My mission is to transform Bella from a technical utility into a meaningful, lasting digital life form.

Key Features
Emotional Intelligence: Features an empathy-driven interaction system capable of understanding context and providing warm, caring feedback.

Personalized Evolution: Employs advanced prompt engineering to allow Bella to develop a unique personality that adapts to your interaction style.

Multimodal Interaction: Integrates voice recognition (Whisper) and visual expression to create a natural, lifelike communication experience.

Extensible Agent Architecture: A highly decoupled, modular design that allows for easy integration of custom tools, APIs, and services.

Usage
Clone and Install

Bash
git clone https://github.com/your-username/Bella-Companion-Assistant.git
cd Bella-Companion-Assistant
npm install
Setup Models

Bash
npm run download
Run

Configure your API keys in the .env file.

Start the service:

Bash
npm start
Access the interface at http://localhost:8081 in your browser.
